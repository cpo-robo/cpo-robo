3 EASY STEPS TO RUN YOUR BOT
============================

STEP 1: SET YOUR NEW API KEY (2 minutes)
----------------------------------------
1. Right-click "SET_API_KEY.bat" and select "Run as administrator"
2. Paste your NEW OpenAI API key (from step above)
3. Press Enter
4. You should see "SUCCESS" message
5. Close the window

STEP 2: START THE BOT (5 seconds)
---------------------------------
1. Double-click "START_SERVER.bat"
2. Wait 2-3 seconds
3. Your browser will open automatically
4. You should see the ARIA interface (dark theme with cyan glows)

STEP 3: CHAT! (Infinite fun)
----------------------------
1. Type a question about Syracuse permits
2. Press ENTER or click SEND
3. ARIA will answer
4. Keep asking!

TROUBLESHOOTING:
===============

Q: "Python is not installed" error
A: You should have Python. If not, install from https://www.python.org

Q: "What's my API key?"
A: Get it from https://platform.openai.com/account/api-keys
   Click "Create new secret key" and copy it

Q: Port 5000 is in use
A: Change the port number in app.py (line 112)
   Change from: port=5000
   To: port=8000

Q: Nothing happens when I click .bat files
A: Right-click and select "Run as administrator"
